AI Writer plugin with OpenRouter integration, real-time writing, and direct publishing.

== Description ==

XU-AIMatic Writer is a powerful AI content generation plugin for WordPress. It integrates with OpenRouter, Google Gemini, and OpenAI to provide high-quality articles directly within your WordPress dashboard.

Features include:
*   Real-time streaming article generation.
*   Auto-image insertion using Pexels, Pixabay, or Pollinations.ai.
*   YouTube video embedding.
*   Campaign manager for bulk automation.
*   SEO-friendly output.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/xu-aimatic-writer-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Navigate to the AIMatic Writer settings page to configure your API keys.
